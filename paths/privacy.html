Privacy Policy

Last updated: January 10, 2026

Solar is a Discord utility bot and filter checker. We value your privacy and are committed to being transparent about how we handle your data.

By using Solar, you agree to this Privacy Policy. If you do not agree with any part of this policy, please discontinue use of the bot immediately.

1. What Data We Collect

  • User Data: Usernames, user IDs, and message content (for commands and features like AFK, snipe, and filters).
  • Server Data: Server (guild) information such as server IDs, member counts, join times, and bot/user breakdowns.
  • Message Metadata: Mentions, deleted messages, and related metadata for bot features.
  • URLs: URLs provided by users for filter checking.

2. How We Use Data

  • Data is used to provide bot features such as AFK status, snipe (showing the last deleted message), filter checking, and server statistics.
  • Deleted messages and AFK statuses are stored only in memory for the duration of the bot’s runtime. This data is lost when the bot restarts and is never written to disk or any external database.
  • For filter checking, only the URLs provided by users are sent to third-party filtering APIs (such as Blocksi, Securly, GoGuardian, etc.) to check their categorization or block status. No Discord user data (like usernames or IDs) is sent to these services—only the URLs being checked.

3. Data Retention

All data (such as AFK statuses and deleted messages) is stored only in memory and is automatically lost when the bot restarts or the data is overwritten. No data is retained long-term or stored externally.

4. Data Sharing

  • Solar does not share any Discord user data with third parties.
  • Only URLs submitted for filter checking are sent to third-party filter APIs. No other data is shared.

5. Your Rights

  • Solar does not store personal data long-term, so there is no data to request, modify, or delete.
  • If you have concerns about privacy or data usage, please contact us using the information below.

6. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted on this page with an updated date.

7. Contact

For questions or concerns about this Privacy Policy, contact the Solar team at contact@solarbot.dev or via our Discord support server.

Thank you for using Solar!
